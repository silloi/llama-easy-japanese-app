# 🗣️ やさしい日本語翻訳ツール

文化庁「在留支援のためのやさしい日本語ガイドライン」に準拠した、通常の日本語を「やさしい日本語」に翻訳するStreamlitアプリです。

## 📋 概要

**やさしい日本語**とは、日本語に不慣れな外国人にもわかりやすい日本語のことです。このツールは、Groq API + Llama 3.3を使用して、通常の日本語を自動的に「やさしい日本語」に変換します。

### 対象ユーザー
- CEFR A2-B1レベルの日本語学習者
- 生活者としての外国人
- 日本での生活に必要な情報を理解したい方

### 主な機能
- ✅ 文化庁ガイドラインに完全準拠
- ✅ Groq API（Llama 3.3）による高速・高品質な変換
- ✅ 5種類のサンプルテキスト（行政・医療・防災・雇用・住宅）
- ✅ Before/After 2カラム表示で比較しやすい
- ✅ シンプルで直感的なUI

## 🚀 セットアップ

### 必要要件
- Python 3.11以上
- Groq APIキー（https://console.groq.com から取得）

### インストール手順

1. **リポジトリのクローン**
```bash
git clone https://github.com/yourusername/llama-easy-japanese-app.git
cd llama-easy-japanese-app
```

2. **依存パッケージのインストール**
```bash
pip install -r requirements.txt
```

3. **環境変数の設定**

Groq APIキーを環境変数に設定します：

```bash
# Linux/Mac
export GROQ_API_KEY="your_groq_api_key_here"

# Windows (PowerShell)
$env:GROQ_API_KEY="your_groq_api_key_here"
```

または、`.env`ファイルを作成：
```
GROQ_API_KEY=your_groq_api_key_here
```

4. **アプリの起動**
```bash
streamlit run app.py

# uvを使用している場合
uv run streamlit run app.py
```

ブラウザで `http://localhost:8501` にアクセスしてください。

## 📖 使い方

1. **テキストを入力**
   - 左側のテキストエリアに変換したい日本語を入力

2. **サンプルを試す（オプション）**
   - 5つのサンプルボタン（行政・医療・防災・雇用・住宅）から選択

3. **変換**
   - 「やさしい日本語に変換」ボタンをクリック
   - 右側に変換結果が表示されます

## 📝 やさしい日本語の主要ルール

このツールは、文化庁ガイドラインの以下のルールに基づいています：

1. **一文一義**: 一つの文には一つの情報だけを書く
2. **文の長さ**: 一文は20〜30文字程度に短くする
3. **箇条書き**: 複数の情報は箇条書きにする
4. **簡単な言葉**: 漢語より和語を使う（例: 実施→する）
5. **説明を添える**: 難しい言葉には簡単な説明を添える
6. **二重否定回避**: 「〜ないことはない」などは使わない
7. **具体的表現**: 「速やかに」→「すぐに」
8. **敬語の簡素化**: 「です・ます」調で統一
9. **カタカナ語注意**: なるべく使わない
10. **時制の明確化**: いつのことか明確にする

## 📊 変換例

### 例1: 行政文書

**原文:**
> 新規の上陸許可を受けて日本に入国した場合、在留カードが交付された方は、住所を定めた日から14日以内に、在留カードをお持ちになってお住まいの市町村において転入の届出をする必要があります。

**やさしい日本語:**
> 日本に入国した人で、在留カードをもらった人は、次のことをしてください。
>
> ・住む場所を決めた日から14日以内に手続きをします
> ・在留カードを持って、住む場所の市役所や区役所に行きます
> ・「転入届」を出します

### 例2: 医療情報

**原文:**
> 発熱や咳などの症状がある場合は、事前に医療機関に電話で相談してから受診してください。マスクを着用し、公共交通機関の利用は控えてください。

**やさしい日本語:**
> 熱や咳があるときは、次のことをしてください。
>
> ・まず、病院に電話で相談します
> ・それから病院に行きます
> ・マスクをつけます
> ・バスや電車は使わないでください
